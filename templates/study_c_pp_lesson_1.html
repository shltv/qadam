If операторы шарт шынайы болса, кейбір кодты орындау үшін қолданылады.

if (condition) {
statements
}

Шарт қандай өрнек есептелетінін анықтайды. Егер шарт шынайы болса, онда фигуралық жақшаға салынған операторлар орындалады.
Егер шарт жалған болса, операторлар елемейді және бағдарлама if операторларының денесінен кейін жұмысын жалғастырады.

Шарттарды бағалау үшін реляциялық операторларды пайдаланыңыз.


Мысалы:

if (7 > 4) {
cout << "Yes";
}
// Output "Yes"

If операторы (7>4) шартын бағалайды, оны шынайы табады және содан кейін cout операторы орындайды.
Егер біз greater операторын less than (7<4) операторына өзгертсек, оператор орындалмайды және ештеңе басып шығарылмайды.
If операторында көрсетілген шарт үтір нүктесін талап етпейді.

Қосымша реляциялық операторлар:



Мысалы:

if (10 == 10) {
cout << "Yes";
}
// Output "Yes"

Not equal to операторы операндаларды есептейді, олардың тең немесе жоқ екенін анықтайды. Егер операндалар тең болмаса, шарт true ретінде есептеледі.
Мысалы:

if (10 != 10) {
cout << "Yes";
}

Жоғарыда көрсетілген шарт false мәнін қабылдайды және Код блогы орындалмайды.

Сіз if операторында айнымалыларды салыстыру үшін реляциялық операторларды пайдалана аласыз.

Мысалы:

int a = 55;
int b = 33;
if (a > b) {
cout << "a is greater than b";
}
// Output "a is greater than b"
